## Anchor::Schema

### Example

```ruby
class MySchema < Anchor::Schema
  resource UserResource
  resource PostResource

  enum UserRoleEnum
end

MySchema.register
```

<Callout>
  Note: Currently, dependent resources and enums do not have their types
  generated. _All_ resources and enums must be registered as part of the schema.
</Callout>

## Class Methods

### `.register`

- Type: [`Anchor::Schema::Register`](#anchorschemaregister)

### `.resource`

- Type: `Anchor::Resource -> void`

<Callout>

`Anchor::Resource` is not defined in Anchor. The definition is:

```ruby
module Anchor::Resource < JSONAPI::Resource
  include Anchor::SchemaSerializable
  abstract
end
```

</Callout>

Adds `resource` to `register`.

### `.enum`

- Type: `Achor::Types::Enum -> void`

Adds `enum` to `register`.
